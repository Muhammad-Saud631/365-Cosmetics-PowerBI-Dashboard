# 365 Cosmetics Analytics Dashboard

## Project Overview

This project is an interactive Power BI dashboard developed for analyzing the business performance of a cosmetics and skincare e-commerce company. The dashboard provides insights into sales performance, profitability, customer behavior, product trends, and regional analysis to support data-driven decision-making.

## Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling

## Key Features

* KPI Reporting Dashboard
* Sales & Profit Analysis
* Customer Segmentation Analysis
* Regional Performance Insights
* Discount Impact Analysis
* Top Products Performance
* Interactive Slicers & Navigation

## Dashboard Pages

1. Landing Page
2. Overview Dashboard
3. Region View
4. Customer View
5. Insights & Recommendations

## Key Insights

* Strong year-over-year sales growth observed from 2020 to 2023.
* Higher discount levels negatively impacted profit margins.
* Top-performing products contributed significantly toward overall profit.
* Certain regions generated high sales but lower profitability.
